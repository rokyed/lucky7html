# Lucky Seven Machine

This small demo shows a slot machine built with HTML, CSS and JavaScript.

The reels use the following classic slot symbols:

🍒 – Cherry
🔔 – Bell
🍋 – Lemon
🍇 – Grapes
🍊 – Orange
7️⃣ – Number 7
💎 – Diamond
⭐ – Star
🍉 – Watermelon
🍀 – Four-leaf clover (lucky symbol)
🃏 – Joker
💰 – Money bag / Jackpot
🎰 – Slot machine
